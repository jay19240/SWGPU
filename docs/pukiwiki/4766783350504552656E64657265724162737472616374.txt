* Class: Gfx3PPERendererAbstract

An abstract post-processing effects renderer.~
- inherit from: Gfx3RendererAbstract~
- parent of: Gfx3PPERenderer~
** Constructors
- ''new Gfx3PPERendererAbstract''(pipelineId: string, vertexShader: string, fragmentShader: string, params: Float32Array): Gfx3PPERendererAbstract~
-- ''pipelineId'': A unique identifier for the render pipeline.~
-- ''vertexShader'': The code for the vertex shader.~
-- ''fragmentShader'': The code for the fragment shader.~
-- ''params'': The params uniform.~
** Methods
- ''getSourceTexture''(): GPUTexture~
Returns the source texture.~
Note: This instance is responsible to create the source texture used to rendering the previous pass.~
This way, it is easy to chain multiple effects.~
~
- ''isEnabled''(): boolean~
Check if post-process effects is enabled.~
~
- ''onWindowResize''(width: number, height: number): void~
Virtual method called on window resize.~
-- ''width'': The width screen.~
-- ''height'': The height screen.~
~
- ''render''(destinationTexture: GPUTexture): void~
The render function.~
-- ''destinationTexture''~
~
- ''setEnabled''(enabled: boolean): void~
Enable post-process effects.~
-- ''enabled'': Indicating whether ppe should be enabled or disable.~
